# Prompt Library

## Rewrite A Paragraph

请使用 $environment-consulting-report-writing-skill 修改以下段落。要求：正式中文环境咨询报告语气；按“分析对象 -> 依据/事实 -> 比较分析 -> 判断 -> 管理含义”组织；保留必要数据、标准和文件依据；避免空泛判断、否定式转折和绝对化表述；输出可直接替换正文的修改稿。

## Review Logic

请使用 $environment-consulting-report-writing-skill 审查以下章节的论证逻辑。重点检查：章节定位、分析边界、法规/标准依据、数据与结论对应关系、同级小节口径一致性、情景或方案比选口径、结论边界、措施可执行性。请输出问题清单和修改建议。

## Condense Text

请使用 $environment-consulting-report-writing-skill 精简以下内容。要求：不改变技术含义；删除重复背景、套话和过度解释；保留分析对象、依据、指标、数据、判断条件和管理要求；语气保持正式、稳健。

## Strengthen Conclusion

请使用 $environment-consulting-report-writing-skill 根据以下材料形成环境咨询报告结论。要求：结论与前文证据对应；写清影响对象、主要依据、判断条件和后续管理要求；避免“总体可控”“无影响”“确保达标”等空泛或绝对化表述。
